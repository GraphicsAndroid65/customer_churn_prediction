🔮 The Intelligent Churn Engine

1. System Architecture & Tech Stack

This full-stack application bridges complex predictive mathematics with a beautiful consumer interface. Hosted efficiently on Google Firebase, this site acts as a decoupled frontend querying a backend predictive engine.

  • Languages: Python 3.10, Vanilla JavaScript, HTML5, CSS3
  • ML Libraries: Scikit-Learn, Pandas, NumPy, Joblib
  • Backend Delivery: FastAPI running async Uvicorn servers
  • Deployment: Firebase Hosting & GitHub Version Control

2. How We Trained The Model

The mathematical heart of this tool uses a Random Forest Classifier. We processed a robust telecommunications dataset containing thousands of real-world permutations involving Tenure, Billing Plans, and Support Escalations. The ensemble method splits the data through multiple isolated decision trees to prevent overfitting and guarantees high precision.

3. Model Statistics

  • Accuracy: Achieved an elite 97.30% Testing Accuracy.
  • Precision: Extracted an F1 Score of 0.98.
  • Feature Importance: "Contract Type" is statistically weighed at ~28.7%, dominating the calculation.

4. Real-World Value & Use Cases

Acquiring a new customer is up to 25x more expensive than retaining an existing one. Our application prevents financial bleeding globally:

  • Telecom & Streaming (Netflix, AT&T): If a user calls support 4 times in a week (high Support Tickets), the ML model immediately flags them as "High Risk" of canceling. The company's automated system instantly emails them a free month or an apology coupon to keep them happy!
  • SaaS B2B Companies: If a firm notices "Monthly" Contract Type customers churn at a 60% higher rate, their dashboard tells the Sales Team to call Monthly customers and offer them an exclusive 20% discount if they upgrade to an annual plan.
  • Saves Millions: By predicting dissatisfaction using Data Science before human employees ever realize it, multi-million dollar corporations proactively inject retention strategies to maximize revenue.
